# AI OS Layers

| Layer | Tool | Responsibility |
|---|---|---|
| Brain（大腦） | ChatGPT | 分析 / Workflow Decision（工作流決策） |
| Workspace（工作區） | Notion | Daily Operation（日常營運） |
| PM（專案） | GitHub | SOP / Workflow / Prompt |
| Automation（自動化） | n8n | Trigger（觸發） / Notification（通知） |
| Knowledge（知識） | Obsidian | Long-term Knowledge（長期知識） |
| Presentation（簡報） | Canva | Proposal（提案） / Slides（簡報） |
