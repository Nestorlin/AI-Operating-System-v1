# AI Apps Routing v1

## Purpose

Define which AI App handles each workflow.

---

# Routing Table

| Workflow | AI App | Purpose |
|---|---|---|
| Meeting Summary | ChatGPT | Meeting analysis |
| Presentation | Canva | AI presentation |
| Task Tracking | GitHub | Issue management |
| Knowledge Storage | GitHub Knowledge | Long-term knowledge |
| Email Workflow | Outlook | Communication |
| PM Workflow | ChatGPT + GitHub | Project management |

---

# Rule

One AI App = One Main Responsibility

Avoid overlapping responsibilities.

---

# Goal

Build scalable AI Operating System workflows.
