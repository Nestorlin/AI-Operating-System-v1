# AI Trigger Rules v1

## Purpose

定義 AI Workflow 觸發規則。

---

# Trigger Rules

| Trigger | Workflow | AI App |
|---|---|---|
| 上傳 Meeting 錄音 | Meeting Workflow | ChatGPT |
| 新增客戶需求 | PM Workflow | ChatGPT + GitHub |
| 建立簡報需求 | Presentation Workflow | Canva |
| 新增知識文件 | Knowledge Workflow | GitHub Knowledge |
| 新增 TODO | PM Tracking Workflow | GitHub |

---

# Workflow Logic

## Meeting Workflow

Meeting Input
↓
Meeting Analysis
↓
Meeting Output
↓
Knowledge Update
↓
GitHub Tracking

---

# Rule

- 一個 Trigger 對應一個主要 Workflow
- 避免重複 Workflow
- 所有重要內容必須更新 Knowledge Base

---

# Goal

建立可自動化 AI Workflow 架構。
